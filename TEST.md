## 시나리오 1: 최초 사용자 (HR)

### 준비
```bash
rm ~/.claude/memory/user-profile.md  # 프로파일 삭제
```

### 실행
1. `/my-day1` 실행
2. 질문 답변:
   - 직업: HR/인사 담당자
   - 전공: 경영/경제
   - 터미널: 처음이에요
3. Block 3-1 선택

### 예상 결과
- [ ] 프로파일 저장 확인: `cat ~/.claude/memory/user-profile.md`
- [ ] Phase A 비유: "회사 인사 규정집처럼..."
- [ ] Phase A 예시: "직원 평가 기준 저장" 등
- [ ] EXECUTE: 터미널 여는 법 포함
- [ ] Phase A 종료 문구 출력
- [ ] Phase B 퀴즈 출제
- [ ] 실무 과제: "인사 업무 규칙 하나를 CLAUDE.md에..."

## 시나리오 2: 재방문 사용자 (개발자)

### 준비
프로파일 수동 생성:
```bash
mkdir -p ~/.claude/memory
cat > ~/.claude/memory/user-profile.md << 'EOF'
# 사용자 프로파일

- 직업: 백엔드 개발자
- 전공: 컴퓨터공학/IT
- 터미널 경험: 익숙해요

생성일: 2026-02-15
EOF
```

### 실행
1. `/my-day1` 실행
2. "네, 이대로 진행합니다" 선택
3. Block 3-1 선택

### 예상 결과

- [ ] Phase A 예시: "코딩 컨벤션 저장" 등
- [ ] EXECUTE: 터미널 여는 법 생략

2. "아니요, 프로파일을 다시 설정하겠습니다" 선택
