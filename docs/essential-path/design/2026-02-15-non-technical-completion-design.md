# Essential Path: 비개발자 완료율 개선 디자인

> **작성일**: 2026-02-15
> **목표**: 개발 지식이 거의 없는 현업 실무자가 my-day1 스킬을 끝까지 완료하도록 개선

---

## 문제 정의

**현재 상황:**
- my-day1 스킬: 13개 블록, 3,604줄 콘텐츠
- 9개 직업군 지원, 3가지 터미널 경험 수준 대응
- Phase A/B 구조로 학습-검증 진행

**비개발자 완료 장벽:**
1. ✅ **완료 기준**: 핵심 블록 7~8개만 완료해도 성공으로 간주해야 함
2. ✅ **최대 장벽**: 기술 용어가 너무 어려움 (CLI, git, MCP 등)
3. ✅ **해결책**: 동기부여 스토리 (실무 사례 제시)
4. ✅ **구조**: 단계별 마일스톤 (중간 성취 배지)

---

## 솔루션: Essential Path 모드

### 핵심 아이디어

13개 블록을 **"필수 7개 ⭐ + 심화 6개 🔷"**로 재구성하여:
- 심리적 부담 감소 (13개 → 7개로 느껴짐)
- 명확한 완료 기준과 성취감 제공
- 실무 사례로 동기부여 강화
- 용어 장벽 제거 (즉석 비유 제공)

## Architecture

### 1. 필수/심화 분류 체계

**필수 미션 7개** (Essential Path):
1. **미션 0**: Setup (Claude Code 설치) - 5분, ⭐ 쉬움
2. **미션 1**: Experience (Working Backward 데모) - 10분, ⭐ 쉬움
3. **미션 3-1**: CLAUDE.md (가장 많이 쓰는 기능) - 10분, ⭐⭐ 보통
4. **미션 3-2**: Skill (반복 작업 자동화) - 15분, ⭐⭐ 보통
5. **미션 3-4**: Subagent (작업 위임) - 10분, ⭐⭐ 보통
6. **미션 요약**: 7개 기능 요약 - 5분, ⭐ 쉬움
7. **미션 4**: Basics (CLI + Git 기초) - 10분, ⭐⭐ 보통

**심화 미션 6개** (Advanced Path):
- 미션 2 (Why), 3-3 (MCP), 3-Break, 3-5 (Agent Teams), 3-6 (Hook), 3-7 (Plugin)

### 2. 마일스톤 시스템

```
🎯 기초 완료 (3개): 미션 0, 1, 3-1 완료 시
   → "Claude Code 사용 준비 완료!"

🏆 핵심 완료 (7개): 필수 7개 완료 시
   → "Claude Code 핵심 마스터! 실무 투입 가능!"

🚀 마스터 (13개): 전체 완료 시
   → "Claude Code 전문가 등극!"
```

### 3. 학습 경로 선택 (스킬 시작 시)

```
어떻게 시작하시겠어요?

1. 빠른 시작 (필수 7개만) ⭐ 추천
   → 핵심만 배우고 바로 실무 적용
   → 예상 시간: 1~1.5시간

2. 전체 과정 (13개 전부)
   → 모든 기능을 깊이 있게 학습
   → 예상 시간: 2~3시간

3. 특정 미션만 선택
   → 필요한 것만 골라서 학습
```

## Components

### 1. 동기부여 스토리 (각 미션 시작 전)

모든 필수 미션 Phase A 시작 시 추가:

```markdown
---
💡 **왜 이 미션을 해야 하나요?**

[직업별 맞춤 스토리]
- HR: "채용 공고 20개를 5분 만에 업데이트하고 싶으신가요?"
- 개발자: "반복되는 코드 리뷰 코멘트를 자동화하고 싶으신가요?"
- 디자이너: "디자인 시스템 문서를 항상 최신으로 유지하고 싶으신가요?"

**실제 사례:**
[구체적인 before/after 시나리오]

**이 미션을 완료하면:**
✓ OO 작업을 자동화할 수 있습니다
✓ OO 시간을 절약할 수 있습니다
---
```

### 2. 용어 설명 시스템 (Inline Glossary)

비개발자가 막히는 주요 용어 15개에 즉석 비유 제공:

| 용어 | 비개발자용 비유 |
|------|----------------|
| CLI | Excel 수식처럼 텍스트로 명령하는 방법 |
| 터미널 | 컴퓨터와 대화하는 텍스트 창 |
| git | Word의 "변경 내용 추적" 강력 버전 |
| commit | 게임 세이브 |
| CLAUDE.md | 회사 규정집 (Claude 전용) |
| Skill | Excel 매크로 (Claude 버전) |
| MCP | 앱 연동 (Slack, Gmail 등) |
| Subagent | 부하 직원에게 일 시키기 |
| Agent Teams | 프로젝트 팀 구성 |
| Hook | "OO하면 자동으로 OO해" 규칙 |
| Plugin | 앱스토어에서 앱 설치하듯 |
| Phase A/B | 1단계: 배우기, 2단계: 퀴즈 |
| push | Google Drive에 업로드 |
| pull | Google Drive에서 다운로드 |
| branch | A안, B안 동시 작업 |

**구현**: 용어 첫 출현 시 자동으로 괄호 안에 비유 삽입

### 3. 마일스톤 배지 표시

각 미션 완료 시:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎯 기초 완료! (3/7 필수 미션)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

완료: 미션 0 ✓, 미션 1 ✓, 미션 3-1 ✓
남은 필수: 미션 3-2, 3-4, 3-Summary, 4

다음: 미션 3-2 (Skill) - 반복 작업 자동화
예상 시간: 10분
```

각 미션에 표시:
- ⭐ 필수 / 🔷 심화 태그
- ⏱️ 예상 소요 시간
- 💪 난이도 표시

## User Flow

### 전체 여정

```
1. 스킬 시작 (/my-day1)
   ↓
2. 프로파일 확인/생성
   ↓
3. 학습 경로 선택 (빠른 시작/전체/커스텀)
   ↓
4. 미션 진행
   각 미션마다:
   - 💡 동기부여 스토리
   - 📖 설명 (용어 즉시 비유)
   - 💻 실습 (막히면 "help")
   - ✅ 퀴즈
   - 🎯 실무 과제
   ↓
5. 마일스톤 배지 획득
   ↓
6. 다음 미션 또는 완료
```

### 막힘 방지 장치

**실습 중 막혔을 때:**
- "help" 또는 "막혔어요" 입력 시 자동 진단 + 해결법 제시
- 화면 캡처 요청 → 문제 파악 → 해결 안내
- "skip" 입력 시 다음 미션으로 (나중에 재방문 가능)

**용어가 어려울 때:**
- "용어" 입력 시 최근 나온 기술 용어 3개 다시 설명
- 더 쉬운 표현 사용 모드 제안

**동기 부족할 때:**
- "왜 해야 해요?" 입력 시 실제 사례 3가지 추가 제시
- Skip 가능하지만 나중 필요성 안내

### 중간 이탈 후 재진입

```
👋 다시 오셨네요!

지난번 여기까지 완료하셨어요:
✓ 미션 0, 1, 3-1 (3/7)

다음 미션: 미션 3-2 (자동화)
예상 시간: 15분

1. 이어서 하기
2. 처음부터 다시
3. 다른 미션 선택
```

## Implementation Changes

### SKILL.md 변경사항

**추가할 섹션:**
1. 스킬 시작 시 "학습 경로 선택" 단계
2. 각 미션에 메타데이터 (⭐필수/🔷심화, ⏱️시간, 💪난이도)
3. 각 필수 미션 Phase A에 "💡 왜 배워야 하나요?" 섹션
4. 용어 자동 비유 변환 로직
5. 마일스톤 배지 표시 로직
6. "help" 명령어 처리 로직

**변경할 부분:**
- 모든 "Block" → "미션" 표현 변경
- 진행 상황 표시를 "미션 3/7" 형식으로
- 온보딩 가이드에 "빠른 시작" 안내 추가

**새로 작성할 콘텐츠:**
- 필수 7개 미션별 동기부여 스토리 (직업별 9가지 × 7미션 = 63개 스토리)
- 용어 사전 15개 × 9개 직업별 비유 = 135개 비유
- 마일스톤 배지 3개 메시지

## Testing & Validation

### 테스트 시나리오

**시나리오 1: 터미널 처음 사용하는 HR**
- 경로: 빠른 시작 (필수 7개)
- 검증: 용어 이해도, 실습 성공률, help 활용도, 완료 의지

**시나리오 2: 터미널 조금 써본 디자이너**
- 경로: 전체 과정 (13개)
- 검증: 동기부여 공감도, 완료율, 심화 미션 이해도

**시나리오 3: 개발 경험 있는 PM**
- 경로: 커스텀 (필요한 미션만)
- 검증: 선택적 학습 편의성, skip 자연스러움

### 성공 기준

- ✅ 필수 7개 완료율 **80% 이상**
- ✅ 중도 이탈률 **20% 이하**
- ✅ "실무에 적용하겠다" 응답 **70% 이상**
- ✅ 용어 이해도 (퀴즈 정답률) **80% 이상**
